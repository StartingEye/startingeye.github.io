## 2025년 2회 4과목: 정보통신 개론

**61. 가상 회선 패킷 교환 방식에 대한 설명으로 옳은 것은?**
1. 수신은 송신된 순서대로 패킷이 도착한다.
2. 우회 경로로 패킷을 전달할 수 있어 신뢰성이 높다.
3. 비연결형 서비스 방식이다.
4. 먼저 전송했더라도 최적의 경로를 찾지 못하면 나중에 전송한 데이터보다 늦게 도착할 수 있다.

> **정답**: 1
> **해설**: 가상 회선(Virtual Circuit) 방식은 데이터를 보내기 전 논리적인 선로 경로를 미리 확립시키고 고정적으로 타고 흐르므로 각 패킷이 **순차적으로 도착함(순서 유지)**이 보장됩니다. 경로가 독립적인 데이터그램 방식에 대한 설명들이 나머치 선지입니다.

<br/>

**62. 정보 통신 시스템의 구성 요소에 대한 용어 설명 중 옳지 않은 것은?**
1. 데이터 회선 종단장치 - DTE
2. 통신 제어장치 - CCU
3. 전단 제어장치 - FEP
4. 신호 변환장치 - DSU

> **정답**: 1
> **해설**: 데이터 회선의 '종단장치'는 신호를 전송로에 적합하게 바꿔주는 모뎀이나 DSU 등의 **DCE(Data Circuit-terminating Equipment)**입니다. 단말기 장비를 의미하는 용어는 DTE(Data Terminal Equipment)입니다.

<br/>

**63. 전송 시간을 일정한 간격의 시간 슬롯(Time Slot)으로 나누고, 이를 주기적으로 각 채널에 할당하는 다중화 방식은?**
1. 주파수 분할 다중화기
2. 파장 분할 다중화기
3. 동기식 시분할 다중화기
4. 회선 분할 다중화기

> **정답**: 3
> **해설**: 전체 전송 가능 대역(시간)을 작게 '타임 슬롯' 단위로 조각내서 배분하는 다중화 방식은 **TDM(Time Division Multiplexing, 통기식 시분할)** 방식입니다.

<br/>

**64. 디지털 데이터 전송에 사용되는 DSU(Digital Service Unit)의 역할은?**
1. 디지털 데이터를 디지털 신호로 변환시킨다.
2. 아날로그 신호를 디지털 데이터로 변환시킨다.
3. 디지털 신호를 아날로그 데이터로 변환시킨다.
4. 아날로그 데이터를 디지털 신호로 변환시킨다.

> **정답**: 1
> **해설**: 전용선을 통해 컴퓨터의 신호를 송신할 때, 컴퓨터 안의 디지털 데이터를 통신 선로에 맞는 **안정적인 디지털 전송 신호(Baseband)** 형태로 상호 변환해 주는 장비가 바로 DSU입니다.

<br/>

**65. OSI 7 계층에 해당하지 않는 것은?**
1. Application Layer
2. Presentation Layer
3. Data Layer
4. Network Access Layer

> **정답**: 4
> **해설**: OSI 참조 모델은 물리(Physical), **데이터 링크(Data Link)**, 네트워크(Network), 전송(Transport), 세션(Session), **표현(Presentation)**, **응용(Application)** 등 7개입니다. Network Access Layer(네트워크 접속 계층)는 TCP/IP 4계층 모델의 하위 모델 명칭입니다.

<br/>

**66. Link State 방식의 라우팅 프로토콜은?**
1. RIP
2. RIP V2
3. IGRP
4. OSPF

> **정답**: 4
> **해설**: 최단 경로(SPF) 알고리즘을 사용하여 전체 네트워크 토폴로지의 정보를 유지하는 링크 상태 파악 기반 라우팅 동적 프로토콜이 바로 **OSPF(Open Shortest Path First)**입니다. RIP 등은 상대적 거리 정보를 쓰는 Distance Vector 방식입니다.

<br/>

**67. 오류가 검출되면 자동적으로 송신 스테이션에게 재전송을 요청하는 ARQ 기법의 형태가 아닌 것은?**
1. Stop-and-Wait ARQ
2. Go-back-N ARQ
3. Responsive-send ARQ
4. Selective-Repeat ARQ

> **정답**: 3
> **해설**: 대표적인 재전송 요구 에러제어(ARQ) 방식으로는 송신 후 확인할 때까지 멈추는 **정지-대기(Stop & Wait)**, 오류가 난 번호부터 전체를 재전송하는 **Go-back-N**, 해당 번호만 재전송하는 **선택적 재전송(Selective-Repeat)** 3가지 모델이 있습니다. 

<br/>

**68. 아날로그 음성 데이터를 디지털 형태로 변환하여 전송하고, 디지털 형태를 원래의 아날로그 음성 데이터로 복원시키는 것은?**
1. CCU
2. DSU
3. CODEC
4. DTE

> **정답**: 3
> **해설**: 음성 등 아날로그 파동 데이터를 코더를 통해 디지털 신호로 PCM 변환하고 이후 디코더를 활용해 원본으로 아날로그화 시키는 그 복합 변환 장치를 **코덱(CODEC)**이라 일컫습니다.

<br/>

**69. 시스템의 회선 구성 방식이 아닌 것은?**
1. 다중화 방식
2. 비동기식 방식
3. 다중점 방식(Multi-Point)
4. 점대점 방식(Point-to-Point)

> **정답**: 2
> **해설**: 다수의 단말 또는 전용 장비가 선로를 어떻게 묶는가 하는 '선로 망 구성 방식'에는 P2P, 다중점 통신, 다중화 장비에 의한 집선 통신 등이 있습니다. 단위가 흐르는 리듬 규칙인 동기/비동기 방식을 지목하는 것은 회선망 형태 구성 문제가 아닙니다.

<br/>

**70. 전송 선로의 조건 중 선로의 감쇠량이 최소로 되는 경우는? (단, R : 선로의 저항, L : 선로의 인덕턴스, C : 선로의 커패시턴스, G : 선로의 누설 컨덕턴스)**
1. RL = GC
2. LG = RC
3. LC = GR
4. LR = CG

> **정답**: 2
> **해설**: 통신 선로에서 신호가 도달할 때 모양의 왜곡이 전혀 일어나지 않고 손실률이 최저가 되는 이른바 헤비사이드(Heaviside)의 이상 무왜곡 조건식은 **$RC = LG$** 입니다.

<br/>

**71. 다음 중 프로토콜의 구성 요소가 아닌 것은?**
1. 재결합(Reassembly)
2. 순서(Timing)
3. 의미(Semantics)
4. 구문(Syntax)

> **정답**: 1
> **해설**: 통신 프로토콜을 성립시키는 3대 핵심 요소는 형식적 포맷을 지정하는 **구문(Syntax)**, 각 기호와 오류의 뜻을 지정하는 **의미(Semantics)**, 그리고 속도 및 시퀀스를 제어하는 **타이밍(Timing, 순서)**입니다.

<br/>

**72. 이동통신 시스템에서 이동체의 움직임에 따라 수신 주파수의 세기가 변하는 현상은?**
1. 동일 채널 간섭
2. 페이딩 현상
3. 열잡음 효과
4. 도플러 효과

> **정답**: 4
> **해설**: 음역이나 전파 발생원 혹은 수신자가 빠르게 상대적으로 움직이고 있을 때, 각 거리에 따라 파장이 압축, 혹은 늘어져서 전달되어 주파수의 고저가 다르게 인식되는 현상을 **도플러 효과(Doppler Effect)**라 칭합니다.

<br/>

**73. 프로토콜의 계층화에 대한 장점이 아닌 것은?**
1. 전체적인 오버헤드(Over Head)가 증가한다.
2. 모듈화에 의한 전체 설계가 쉽다.
3. 이기종 간의 호환성 유지가 쉽다.
4. 한 계층을 수정할 때 다른 계층에 영향을 주지 않는다.

> **정답**: 1
> **해설**: 부품 모듈처럼 통신 구조 층을 나누었기에 개발 수정과 제조사 간 호환성이 강점입니다. 그러나 이를 구현하기 위해 불필요한 공통 데이터 부가(헤더)가 붙거나 통신 연산 부하, 즉 오버헤드가 **증가**한다는 사실은 단점이자 한계이므로 도입에 따른 장점으로 볼 수 없습니다.

<br/>

**74. TCP 전송 계층 프로토콜을 사용하여 통신하는 데 이용되는 소켓을 무엇이라 하는가?**
1. 스트림 소켓
2. 데이터그램 소켓
3. raw 소켓
4. 리시빙 소켓

> **정답**: 1
> **해설**: 네트워크 프로그래밍에서 지향적이고 지속 연결 기반의 신뢰성을 지닌 TCP 프로토콜을 담보하는 방식 소켓을 **스트림(Stream) 소켓**이라 합니다. 무가공/비연결 UDP 체계는 데이터그램(Datagram) 소켓입니다.

<br/>

**75. 다음 중 패킷 교환망의 주요 기능과 거리가 먼 것은?**
1. 논리 채널
2. 트래픽 제어
3. 인터페이스
4. 오류 제어

> **정답**: 3
> **해설**: 패킷 단위로 분할된 정보를 목적지까지 가변적으로 교환망 서버들이 컨트롤할 때 요구되는 네트워크 내부 서비스는 다중화(논리 채널), 오류 제어, 흐름제어, 체증, 라우팅(트래픽 제어)입니다. 단말 접속 연결성(인터페이스)은 물리 등 끝단의 범주입니다.

<br/>

**76. 화상 정보가 축적된 정보센터의 데이터베이스를 수신기와 공중 전화망에 연결해서 이용자가 TV 화면을 보면서 상호대화 형태로 각종 정보 검색을 할 수 있는 것은?**
1. Teletext
2. Videotex
3. HDTV
4. CATV

> **정답**: 2
> **해설**: TV 방송 회선 대신 전화선을 매개로 데이터베이스에 가입 접속한 후, 리모컨 등 키보드를 이용해 시청자가 주도적으로 메뉴를 선택해 대화형 그래픽 데이터를 표출받는 원시 다중 뉴미디어 양방향 시스템을 **비디오텍스(Videotex)**라 합니다.

<br/>

**77. 종합 정보 통신망(ISDN)에 대한 설명으로 잘못된 것은?**
1. 단일 통신망으로 음성 및 비음성 서비스를 포함한 다양한 서비스를 종합적으로 제공한다.
2. 통신 방식은 디지털이나 전송로가 아날로그이므로 중간에 모뎀(MODEM)이 사용된다.
3. 64Kbps의 회선 교환 서비스를 기본으로 제공한다.
4. 통신망의 교환 접속 기능에는 회선 교환 방식과 패킷 교환 방식이 있다.

> **정답**: 2
> **해설**: ISDN은 이름 뜻(Integrated Services 'Digital' Network) 그대로 송신 단말에서 지역 교환기, 그리고 목적지까지 **완벽한 디지털 전송로 체계**로 연결되어 있기 때문에 원래의 아날로그 변복조 장치인 모뎀을 필요로 하지 않습니다.

<br/>

**78. 수신단에서 패리티 체크(Parity Check)를 하는 주된 목적은?**
1. 기억 장치의 용량 검사
2. 전송된 부호의 오류 검사
3. 전송된 데이터의 용량 검사
4. 검출된 오류를 정정

> **정답**: 2
> **해설**: 송신자가 홀수/짝수 단위로 맞추어 보내놓은 부가 비트 값인 패리티 코드를 수신 측이 다시 계산해서 대조해 봄으로써, **통신 간 발생한 오류(비트 반전)**를 찾아내려는(검출) 목적입니다. 단순 패리티 체크는 오류의 위치 지정 및 스스로 정정(Correction)은 불가능합니다.

<br/>

**79. HDLC 프레임의 구조가 순서대로 옳은 것은?**
1. 플래그 → 주소부 → 제어부 → 정보부 → FCS → 플래그
2. 플래그 → 제어부 → 정보부 → 주소부 → FCS → 플래그
3. 플래그 → 주소부 → 정보부 → 제어부 → FCS → 플래그
4. 플래그 → 제어부 → 주소부 → 정보부 → FCS → 플래그

> **정답**: 1
> **해설**: 데이터 링크 표준 제어 프로토콜인 HDLC의 패킷 껍질 기본 프레임 구조는 시작 구별자(**플래그**) 바로 뒤에 대상 송수신지 **주소(A)** 필드, 흐름을 담당하는 **제어(C)** 필드, 비가공 데이터인 **정보(I)** 부, 그리고 에러 유무 확인을 위한 **FCS 필드**와 끝 종료 **플래그(F)**로 덮입니다.

<br/>

**80. 광대역 종합 정보 통신망인 ATM 셀(Cell)의 구조로 옳은 것은?**
1. Header : 5 옥텟, Payload : 53 옥텟
2. Header : 5 옥텟, Payload : 48 옥텟
3. Header : 2 옥텟, Payload : 64 옥텟
4. Header : 6 옥텟, Payload : 52 옥텟

> **정답**: 2
> **해설**: 비동기 전송 방식(ATM) 통신의 최소 공통 전송 단위 덩어리인 하나의 '셀(Cell)'은 총 53바이트(옥텟) 규격이며, 송수진 데이터 제어가 담긴 머리 부분 **헤더 5바이트**와 순수 데이터가 담긴 **페이로드(Payload) 영역 48바이트**로 쪼개져 고정적으로 존재합니다.

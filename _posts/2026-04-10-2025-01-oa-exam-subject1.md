## 2025년 1회 1과목: 사무자동화 시스템

**1. 운영체제에 관한 설명 중 가장 옳지 않은 것은?**
1. 프로세서, 메모리, 입출력장치 등의 하드웨어 자원을 관리한다.
2. 컴퓨터(PC)나 스마트폰과 같은 하드웨어 장치와 사용자 사이의 인터페이스 역할을 한다.
3. 응용 프로그램들은 운영체제가 제공하는 기능에 의존하여 시스템 자원에 접근한다.
4. 응용 프로그램의 유지 보수를 담당하기 때문에 응용 소프트웨어라고도 한다.

> **정답**: 4
> **해설**: 운영체제는 컴퓨터 시스템의 하드웨어와 소프트웨어 자원을 효율적으로 관리하는 **시스템 소프트웨어**입니다. 응용 소프트웨어가 아니며, 사용자가 응용 프로그램을 쉽게 실행할 수 있는 환경을 제공합니다.

<br/>

**2. 다음 중 입력장치에 해당하지 않는 것은?**
1. Plotter
2. Mouse
3. Keyboard
4. Scanner

> **정답**: 1
> **해설**: 플로터(Plotter)는 도면, 현수막, 설계도 등을 출력할 때 사용하는 대형 **출력장치**입니다.

<br/>

**3. 다음 중 사무자동화의 기대 효과와 가장 거리가 먼 것은?**
1. 조직의 최적화
2. 생산성의 개선
3. 부수적 기능(Shadow Function)의 증가
4. 적시성(Timing)의 증가

> **정답**: 3
> **해설**: 사무자동화는 중복되거나 불필요한 부수적 업무(Shadow Function)를 **감소**시키고 핵심적인 주 업무에 집중할 수 있도록 돕는 것이 목적입니다.

<br/>

**4. 다음 중 팩시밀리에서 사진, 문자, 그림 등을 정해진 방식으로 다수의 화소로 분해하는 과정은?**
1. 반사
2. 변조
3. 주사
4. 동기

> **정답**: 3
> **해설**: **주사(Scanning)**는 송신할 원고의 화상 정보를 빛의 반사 원리를 이용하여 전기적 신호(화소)로 잘게 분해하여 읽어내는 과정입니다.

<br/>

**5. 다음 중 CALS의 개념에 포함되지 않는 것은?**
1. EDI(Electronic Data Interchange)
2. BPR(Business Process Reengineering)
3. ECR(Efficient Consumer Response)
4. MPC(Multimedia Personal Computer)

> **정답**: 4
> **해설**: CALS(광속거래)는 제품의 수명 주기 전반에 걸친 모든 정보를 통합, 디지털화하여 기업 내·외부간 공유하는 시스템입니다. EDI, BPR, 전자상거래 등과 밀접하지만, MPC(멀티미디어 PC) 표준 규격과는 거리가 멉니다.

<br/>

**6. 데이터의 생성 양, 주기, 형식 등이 기존 데이터에 비해 매우 크기 때문에, 종래의 방법으로는 수집·저장·검색·분석이 어려운 방대한 데이터를 의미하는 개념은?**
1. 빅 데이터
2. 데이터 마트
3. 데이터 웨어하우스
4. 네트워크 데이터베이스

> **정답**: 1
> **해설**: 기존 데이터베이스 관리 도구의 능력을 넘어서는 대량(Volume), 빠른 속도(Velocity), 고도의 다양성(Variety)을 갖춘 정보 자산을 **빅데이터(Big Data)**라고 합니다.

<br/>

**7. 특정 기업 간의 CALS 및 EDI를 통한 수주, 구매, 조달 및 납품 등과 관련된 기업 간의 전자상거래를 의미하는 용어로 가장 적합한 것은?**
1. B2C
2. B2B
3. C2C
4. C2G

> **정답**: 2
> **해설**: 기업과 기업 간에 이루어지는 전자상거래 방식은 **B2B(Business to Business)**입니다.

<br/>

**8. 집중 처리 시스템의 과도한 처리 집중에 따른 문제점이 아닌 것은?**
1. 통신관리상의 문제점
2. 운용상의 문제점
3. 시스템 사용상의 문제점
4. 사용자 응용 업무 개발 시 문제점

> **정답**: 1
> **해설**: 집중 처리 시스템은 중앙의 메인프레임 하나에 데이터 처리가 쏠리므로 시스템 운용이나 사용상 병목이 생길 수 있으나, 통신망이나 분산 데이터베이스 관리에 비하면 **통신 및 정보 관리의 복잡성** 자체는 오히려 분산 시스템 환경에서 더 문제시되는 경향이 있습니다.

<br/>

**9. 그룹웨어 시스템의 특징에 관한 사항으로 옳지 않은 것은?**
1. 공동 작업이나 공동 목표에 참여하는 다양한 작업 그룹을 지원한다.
2. 신속하고 정확한 의사결정을 지원하는 시스템이다.
3. 개인 작업만의 생산성 향상을 주목적으로 하고 있다.
4. 클라이언트/서버 환경에서 많이 구현된다.

> **정답**: 3
> **해설**: 그룹웨어는 다수의 구성원이 협업을 통해 업무 효율을 높이는 소프트웨어이므로, 개인만이 아닌 **조직(그룹) 전체의 생산성 향상**이 주 목적입니다.

<br/>

**10. HDD와 같은 S-ATA 인터페이스를 사용하고 기계적 부품이 아닌 반도체를 기반으로 제작되어 기존 하드디스크를 대체할 수 있는 저장장치는?**
1. Blu-ray
2. SSD
3. WORM
4. RAM

> **정답**: 2
> **해설**: **SSD(Solid State Drive)**는 플래시 메모리 반도체를 이용하여 HDD에 비해 속도가 빠르고 소음이 없으며 전력 소모가 적어 최신 PC 저장장치로 널리 쓰입니다.

<br/>

**11. 사무자동화(OA)의 접근방법의 유형에 속하지 않는 것은?**
1. 부분 전개 접근방법
2. 업무별 접근방법
3. 사원별 접근방법
4. 계층별 접근방법

> **정답**: 3
> **해설**: 사무자동화의 도입 접근 방식에는 전사적 접근, 부분 전개(부서별) 접근, 기기별 접근, 업무별 접근, 계층별 접근 등이 있으나, 개인 **사원별 접근방법은 없습니다**.

<br/>

**12. 기업의 내부, 외부의 비즈니스, 데이터를 수집·가공하고 관리자들에게 필요한 때에 요구하는 정보를 곧바로 제공할 수 있는 시스템은?**
1. SIS
2. MIS
3. POS
4. DSS

> **정답**: 2
> **해설**: 관리자들의 의사결정과 기업 경영 관리를 지원하기 위해 각종 데이터를 종합적으로 제공하는 시스템을 **MIS(Management Information System, 경영정보시스템)**라고 합니다.

<br/>

**13. 사무자동화의 환경 개선이 주는 효과로 옳은 것은?**
1. 사무기기의 고장 시간의 증가
2. 정확성 및 생산성의 저하
3. 권태감 등의 감소로 작업자의 사기 고취
4. 사무자동화 기기 위주의 배려로 작업 능률 향상

> **정답**: 3
> **해설**: 쾌적한 사무환경은 작업자의 스트레스와 피로도를 줄여주고, 권태감을 감소시켜 사기를 높이고 생산성을 증진시킵니다. 물리적 기기 위주가 아닌 '인간 중심'의 배려가 중요합니다.

<br/>

**14. 팩시밀리의 특징으로 틀린 것은?**
1. 일반 전화회선을 이용하여 즉시 전송 가능하다.
2. 종이 원고의 내용을 원격지에서 충실하게 기록 재생할 수 있다.
3. 원하는 시간에 원하는 정보 전송이 가능하다.
4. 동일 내용을 한 번에 한 명의 수신자에게만 보낼 수 있다.

> **정답**: 4
> **해설**: 현대의 팩시밀리는 다수의 수신자에게 동일한 문서를 동시에 전송하는 **동보 전송 기능**을 지원합니다.

<br/>

**15. 전자우편을 엽서가 아닌 밀봉된 봉투에 넣어서 보낸다는 개념으로 IETF에서 인터넷 초안으로 채택한 것은?**
1. PEM
2. PGP
3. S/MIME
4. PGP/MOME

> **정답**: 1
> **해설**: **PEM(Privacy Enhanced Mail)**은 인터넷 상에서 비밀성, 메시지 무결성, 사용자 인증 등을 제공하는 보안 이메일 표준입니다.

<br/>

**16. 데이터를 복수 또는 분할 저장하여 병렬로 데이터를 읽는 보조기억장치 또는 그 방법으로, 디스크의 고장에 대비하여 데이터의 안정성을 높이는 기술은?**
1. SASD
2. DASD
3. RAID
4. NAC

> **정답**: 3
> **해설**: **RAID(Redundant Array of Independent Disks)** 다수의 하드디스크를 묶어 하나의 논리적 디스크로 구성하여, 속도를 높이거나 데이터 안정성을 극대화하는 스토리지 기술입니다.

<br/>

**17. 모니터 등의 디스플레이나 프린터의 해상도 단위이며 1 inch당 몇 개의 dot(점)가 들어가는지를 말하는 것은?**
1. DPI
2. BPS
3. PPT
4. LPM

> **정답**: 1
> **해설**: **DPI(Dots Per Inch)**는 인쇄나 화면 출력의 해상도 단위로, 가로나 세로 1인치 안에 몇 개의 점(Dot)이 들어가는지를 의미합니다.

<br/>

**18. 데이터베이스의 모형에 속하지 않는 것은?**
1. 정규형 데이터베이스
2. 네트워크형 데이터베이스
3. 계층형 데이터베이스
4. 관계형 데이터베이스

> **정답**: 1
> **해설**: 대표적인 데이터베이스 논리적 모델(모형)에는 계층형(Hierarchical), 망형(네트워크형, Network), 관계형(Relational), 객체지향형 등이 있습니다. '정규형'은 데이터베이스를 설계할 때 이상현상을 없애기 위한 구조화 과정을 뜻합니다.

<br/>

**19. 사무자동화로 기대되는 정성적 효과가 아닌 것은?**
1. 의사결정을 신속히 할 수 있다.
2. 업무처리시간을 단축할 수 있다.
3. 정보획득 시간이 길어진다.
4. 개인의 업무처리 능력이 향상된다.

> **정답**: 3
> **해설**: 자동화를 통해 문서 및 정보 검색이 쉬워지므로 정보획득에 걸리는 시간이 **획기적으로 짧아집니다**.

<br/>

**20. 정보화시대의 자동화의 분류 중 설명이 가장 바르게 표현된 것은?**
1. 사무자동화 : POS 시스템을 기본으로 한 스토어 컨트롤러로 구성
2. 빌딩자동화 : 가전기기운용, 방범 방재 및 비디오폰 기능의 구성
3. 공장자동화 : 부분적인 자동화, 단위기계의 완전자동화, 생산 라인의 자동화 시스템
4. 점포자동화 : 각종 설비를 원격 제어, 감시하는 시스템

> **정답**: 3
> **해설**:
- POS 시스템 중심 = 점포자동화
- 가전기기 운용, 방범 시스템 등 = 가정자동화(Home Automation)
- 설비 원격 감시 및 공조 제어 = 빌딩자동화
따라서 3번 공장자동화(FA)에 대한 설명이 가장 정확합니다.

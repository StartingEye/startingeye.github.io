## 2025년 1회 4과목: 정보통신 개론

**61. 다음 중 정보 통신 시스템의 특징이 아닌 것은?**
1. 정보 처리의 효율화
2. 대용량 파일의 공동 이용
3. 대형 컴퓨터의 공동 이용
4. 거리와 시간의 극복

> **정답**: 1
> **해설**: 정보 통신 시스템(Data Communication System)은 정보를 원격으로 시공간의 제약 없이 신속하게 주고받기 위해 구성하므로 거리/시간 극복, 파일 공동 이용, 컴퓨터 분산 이용이 주된 이점입니다. '정보 처리의 효율화' 자체는 컴퓨터 내부적인 데이터 프로세싱 시스템의 주된 목표에 더 가깝습니다.

<br/>

**62. 다음 중 데이터 회선 종단장치와 관련이 없는 것은?**
1. DCE
2. DTE
3. MODEM
4. DSU

> **정답**: 2
> **해설**: 데이터 회선 종단장치(DCE)에 속하는 것은 모뎀(MODEM)이나 디지털 서비스 장치(DSU) 등입니다. 반대로 사용자가 데이터를 직접 조작하는 맨 끝단 단말 장치를 **DTE(Data Terminal Equipment)**라고 부릅니다.

<br/>

**63. 정보 통신 시스템이 수행하는 처리 방식으로 틀린 것은?**
1. 원격 일괄 처리 방식
2. 주파수 분할 처리 방식
3. 시간 공유 처리 방식
4. 거래 처리 방식

> **정답**: 2
> **해설**: 일괄 처리(Batch), 시간 분할 처리(Time Sharing), 실시간/거래 처리(Transaction)는 정보 처리 시스템의 운용 방식입니다. **주파수 분할**은 한 회선을 여러 대역폭으로 나누는 통신선의 다중화(Multiplexing) 방식입니다.

<br/>

**64. 패킷망에서 데이터의 양이 적고, 융통성이 요구되는 경우에 가장 적합한 교환방식은?**
1. 회선 다중통신(Circuit Multiplexing)
2. 가상 회선(Virtual Circuit)
3. 메시지 교환(Message Switching)
4. 데이터그램(Datagram)

> **정답**: 4
> **해설**: 패킷 교환 방식에는 통신 전 미리 경로를 배정하는 '가상 회선'과 각 패킷이 최적 경로를 독자적으로 찾아 목적지로 전송되는 **데이터그램** 방식이 있습니다. 데이터 양이 적거나 트래픽 변동이 잦을 때는 경로 설정 없이 바로 보내는 데이터그램 방식이 유연성(융통성)에서 적합합니다.

<br/>

**65. 다음 중 16진 QAM의 대역폭 효율은?**
1. 3 bps/Hz
2. 4 bps/Hz
3. 5 bps/Hz
4. 6 bps/Hz

> **정답**: 2
> **해설**: 16-QAM(직교 진폭 변조)은 서로 다른 진폭과 위상의 조합으로 한 번호(심볼)당 16개의 신호 상태를 나타낼 수 있습니다. $2^4 = 16$이므로, 한 번호당 4개의 비트(4 bps/Hz)를 전송합니다.

<br/>

**66. LAN의 한 종류인 100Base-T 네트워크에서 사용되는 전송 매체는?**
1. Coaxial cable
2. Optical cable
3. UTP cable
4. Microwave cable

> **정답**: 3
> **해설**: '100Base-T'에서 T는 **Twisted pair(꼬임선)** 케이블을 의미합니다. 이는 일상적으로 많이 쓰는 LAN 케이블인 **UTP(Unshielded Twisted Pair) 케이블**입니다.

<br/>

**67. 아날로그 데이터를 디지털 신호로 변환하는 대표적인 PCM(Pulse Code Modulation) 변조 방식의 과정은?**
1. 표본화 → 양자화 → 부호화 → 복호화
2. 표본화 → 부호화 → 복호화 → 양자화
3. 표본화 → 부호화 → 양자화 → 복호화
4. 표본화 → 복호화 → 부호화 → 양자화

> **정답**: 1
> **해설**: 아날로그 음성/데이터를 디지털화하는 송신측의 과정은 **표본화(Sampling) -> 양자화(Quantization) -> 부호화(Encoding)** 이며, 수신측에서 원음으로 복원해 내는 과정이 복호화(Decoding)입니다.

<br/>

**68. ITU-T 권고안의 X 시리즈에서 패킷형 DTE와 DCE 간의 인터페이스는?**
1. X.21
2. X.22
3. X.24
4. X.25

> **정답**: 4
> **해설**: **X.25**는 패킷 교환망(PSDN) 접속을 위한 표준 프로토콜로, 단말(DTE)과 망 종단(DCE) 간의 패킷 전송 규칙 및 인터페이스를 정의합니다.

<br/>

**69. L2 스위치의 기본 기능이 아닌 것은?**
1. Address Learning
2. Filtering
3. Forwarding
4. Routing

> **정답**: 4
> **해설**: OSI 2계층 MAC 주소를 기반으로 동작하는 스위치는 맥 단위의 학습(Learning), 필터링, 포워딩 기능을 수행합니다. 대역을 넘어 **IP 패킷 경로를 설정해주는 라우팅(Routing)**은 3계층 장비(L3 스위치/라우터)의 기능입니다.

<br/>

**70. 다음 중 전송 과정에서 발생하는 오류의 주요 원인이 아닌 것은?**
1. 백색 잡음
2. 지연 왜곡
3. 누화 잡음
4. 패리티 비트

> **정답**: 4
> **해설**: 백색 잡음이나 왜곡, 신호 간섭인 누화는 회선 상 오류(Error)의 발생 원인입니다. 반면 **패리티 비트(Parity Bit)**는 시스템이 오류가 발생했는지 검사하기 위해 인위적으로 삽입하는 검출용 추가 비트입니다.

<br/>

**71. 에러가 발생되지 않는 이상적인 통신로(무잡음 이산 채널)의 채널 용량은? (단, C : 채널용량, n개의 기호들은 동일 확률을 가지고 있다.)**
1. C = log2(n-2)
2. C = log2(n)
3. C = (n-1)log2(n)
4. C = n * log2(1/n)

> **정답**: 2
> **해설**: 기호가 발생할 확률이 $p = 1/n$ 로 균등할 경우, 가능한 최대 정보 전송 용량(엔트로피)은 $C = \log_2 n$ 비트로 나옵니다.

<br/>

**72. OSI 7계층의 네트워크 계층 기능과 관계가 있는 것은?**
1. 경로 선택
2. 오류의 검출 및 복구
3. 프레임의 순서 제어
4. 프레임의 시작과 끝을 구분

> **정답**: 1
> **해설**: 네트워크(Network, 3계층)의 핵심 기능은 데이터(패킷)가 목적지까지 최적으로 갈 수 있도록 경로를 설정하는 **라우팅(경로 선택)**입니다. 오류 제어 및 프레임 관리는 2계층인 데이터 링크 계층 영역입니다.

<br/>

**73. 전송 속도가 9,600[bps]인 데이터를 8진 PSK로 변조하여 전송할 때 변조 속도는 몇 [Baud]인가?**
1. 1,600
2. 2,400
3. 3,200
4. 4,800

> **정답**: 3
> **해설**: 8-PSK는 각 상태가 8가지이므로 한 번에 $\log_2 8 = 3$ 비트씩 실어 나릅니다. 전송속도(bps) = 변조속도(Baud) × 한 신호당 비트 수. 
> 즉 9,600 = Baud × 3이므로, 변조 속도(Baud)는 9,600 / 3 = **3,200 [Baud]** 입니다.

<br/>

**74. 다음 중 정보의 전달 체계를 무엇이라 하는가?**
1. 통신 제어망
2. 정보 통신망
3. 교환장치
4. 단말장치

> **정답**: 2
> **해설**: 멀리 떨어져 있는 장치 구역 사이에서 전기적인 신호 등을 통해 유무선으로 정보를 교환하고 전달하는 물리적/논리적 연결 망 체계를 **정보 통신망(Information Communication Network)**이라고 부릅니다.

<br/>

**75. DNS 서버가 사용하는 TCP/UDP 포트 번호는?**
1. 11
2. 26
3. 53
4. 104

> **정답**: 3
> **해설**: 도메인 네임을 IP로 바꾸거나 처리해 주는 DNS(Domain Name System)는 잘 알려진(Well-Known) 포트 **53번**을 기본으로 사용합니다.

<br/>

**76. LAN의 채널 액세스 제어 방식인 CSMA/CD에 대한 내용으로 잘못된 것은?**
1. 모든 제어기는 차등의 액세스 권리를 갖는다.
2. 제어기가 상위 계층으로부터 패킷을 받고 나서 전송을 완료할 때까지의 시간은 확률적으로 변화한다.
3. 각 제어기는 액세스 제어에 관하여 반송파 검출 및 충돌 검출만 필요하다.
4. 채널로 송출된 패킷은 모든 제어기에서 수신 가능하다.

> **정답**: 1
> **해설**: CSMA/CD(Carrier Sense Multiple Access with Collision Detection)를 사용하는 버스 형태의 네트워크에서는 허브/스위치에 물려있는 **모든 노드(제어기)가 동등한 회선 엑세스(전송) 권한**을 갖습니다.

<br/>

**77. 다음 중 무선 랜(Wireless LAN)의 표준 규격으로 옳은 것은?**
1. IEEE 802.1
2. IEEE 802.3
3. IEEE 802.11
4. IEEE 802.15

> **정답**: 3
> **해설**: 통신 규약 국제기구인 IEEE이 권고한 무선 근거리 통신망(Wi-Fi 무선 LAN) 국제 표준 번호는 **IEEE 802.11** 묶음 규격입니다. 참고로 802.3은 유선 이더넷, 802.15는 블루투스 등 WPAN 표준입니다.

<br/>

**78. 통신 소프트웨어의 세 가지 기본 구성요소로 옳은 것은?**
1. 데이터 송·수신, 통신 하드웨어 제어, 이용자 인터페이스 제어
2. 데이터 입·출력 제어, 데이터 처리, 데이터 분배
3. 네트워크 제어, 전송 부호 관리, 이용자 인터페이스 제어
4. 데이터 입·출력 제어, 데이터 전송 제어, 통신 회선 제어

> **정답**: 1
> **해설**: 단말에 설치되는 통신 소프트웨어(프로그램) 기능의 3대 구조는 하드웨어를 다루는 **통신 하드웨어 제어**, 프로토콜에 맞춘 **데이터의 송·수신 처리**, 그리고 사용자가 조작하기 쉽도록 구성하는 **이용자 인터페이스 제어** 기능입니다.

<br/>

**79. Shannon의 표본화 정리에 의하면 보내려는 신호 성분 중 최고 주파수의 최소 몇 배 이상으로 표본을 행하면 원 신호를 충실하게 재현시킬 수 있는가?**
1. 1
2. 2
3. 4
4. 8

> **정답**: 2
> **해설**: 샤논 또는 나이퀴스트(Nyquist)의 정리에 따르면, 정보 전파나 표본화 시 **가장 높은 최고 주파수가 갖는 파형 횟수의 최소 2배 이상**의 속도로 표본화를 거치면 간섭 없이 나중에 원 신호로 재구축이 가능합니다.

<br/>

**80. 디지털 변조에서 디지털 데이터를 아날로그 신호로 변환시키는 키잉(Keying) 방식에 해당하지 않는 것은?**
1. 스펙트럼 편이 키잉
2. 진폭 편이 키잉
3. 주파수 편이 키잉
4. 위상 편이 키잉

> **정답**: 1
> **해설**: 모뎀에서 수행하는 디지털-아날로그 간 주요 변조 기술(Keying)로는 파장의 높낮이를 조정하는 **ASK(진폭 편이 키잉)**, 파장의 빈도를 조정하는 **FSK(주파수 편이 키잉)**, 파장의 시작점 각도를 비트는 **PSK(위상 편이 키잉)**가 있습니다. 스펙트럼 편이 키잉이라는 용어는 없습니다.

## 2025년 2회 1과목: 사무자동화 시스템

**1. 다음 중 그룹웨어의 기능과 거리가 먼 것은?**
1. 의사결정 기능
2. 이미지 편집 기능
3. 정보공유 기능
4. 업무흐름 관리 기능

> **정답**: 2
> **해설**: 그룹웨어(Groupware)는 다수의 구성원이 협업하도록 돕는 소프트웨어로 정보 공유, 의사결정 지원, 작업 절차(업무흐름) 관리가 주된 기능입니다. 이미지 편집은 그래픽 소프트웨어의 영역입니다.

<br/>

**2. 다음 중 데이터베이스 시스템에 대한 설명으로 옳지 않은 것은?**
1. 하나의 데이터를 여러 분야에서 공통적으로 사용할 수 있다.
2. 파일 시스템에 비해 상대적으로 파일 수정이 용이하다.
3. 데이터 중복이나 분산에 의한 오버헤드(Overhead)를 줄일 수 있다.
4. 파일이 파괴되었을 때 복구가 빠르고 완벽하다.

> **정답**: 4
> **해설**: 데이터베이스 시스템은 모든 정보가 중앙에 집중되어 있기 때문에 고장이나 파일 파괴가 발생했을 경우 시스템 복구가 시스템 파일 방식보다 훨씬 복잡하고 어렵습니다.

<br/>

**3. 프로그램의 실행 중 인터럽트(Interrupt)가 발생할 경우에 현재의 프로그램 상태가 저장되어 있는 레지스터를 무엇이라 하는가?**
1. PSW
2. PC
3. PCW
4. ACC

> **정답**: 1
> **해설**: 인터럽트 등이 발생했을 때 중앙처리장치의 시스템 상태(플래그 정보 확인 등)와 다음 실행 조건 등을 저장하는 레지스터는 **PSW(Program Status Word)**입니다.

<br/>

**4. 자동화의 체계 중 사무관리의 자동화가 아닌 것은?**
1. 물건 생산과 유통의 자동화
2. 정보의 작성과 정보처리의 자동화
3. 자금의 운용과 유통의 자동화
4. 프로그램의 작성 자동화

> **정답**: 4
> **해설**: 프로그램의 작성 자동화는 소프트웨어 공학 도구(CASE 등)에 의한 개발의 자동화 영역이며, 일반적인 사무관리 및 경영(생산, 정보, 자금 등) 시스템 자동화 범주와는 다소 성격이 다릅니다.

<br/>

**5. 데이터를 복수 또는 분할 저장하여 병렬로 데이터를 읽는 보조기억장치 또는 그 방법으로, 디스크의 고장에 대비하여 데이터의 안정성을 높이는 기술은?**
1. SASD
2. DASD
3. RAID
4. NAC

> **정답**: 3
> **해설**: 디스크 배열을 구성하여 여러 디스크에 데이터를 분산 또는 복제 보관함으로써 성능과 안정성을 극대화한 스토리지 기술이 **RAID**입니다.

<br/>

**6. 의사결정에 필요한 정보를 데이터베이스로부터 검색하여 필요한 분석을 하고 보기 쉬운 형태로 편집출력해 주는 시스템의 개념에 해당하는 용어로 가장 옳은 것은?**
1. CALS
2. DSS
3. MIS
4. PERTS

> **정답**: 2
> **해설**: 관리자가 중요한 의사결정을 내릴 수 있도록 계량적 모델링이나 데이터 분석 정보를 제공하며, 그 결과를 가공, 출력해 조언하는 정보 시스템은 **DSS(Decision Support System, 의사결정지원시스템)**입니다.

<br/>

**7. 객체 지향형 데이터베이스 시스템에서 특정 데이터 구조와 메소드(Method)들로 구성된 객체들의 모임은?**
1. 애트리뷰트(Attribute)
2. 클래스(Class)
3. 릴레이션(Relation)
4. 튜플(Tuple)

> **정답**: 2
> **해설**: 공통적인 성긴 데이터 구조와 행위(메서드)를 가진 객체(Object)들의 모임 또는 그 설계도를 객체지향에서는 **클래스(Class)**라고 부릅니다.

<br/>

**8. 컴퓨터 시스템에 사용되는 메모리에 대한 설명으로 가장 거리가 먼 것은?**
1. RAM은 일반적으로 주기억 메모리(Main Memory)로 사용되며, 휘발성이 없다.
2. 캐시 메모리는 CPU와 기억장치 간의 속도 차를 해소하기 위한 메모리이다.
3. 가상 메모리는 보조기억장치의 일부를 주기억장치처럼 사용하기 위한 메모리이다.
4. 버퍼 메모리는 2개의 장치가 데이터를 주고 받을 때 두 장치 간의 속도 차를 해소하기 위한 장치이다.

> **정답**: 1
> **해설**: RAM(Random Access Memory)은 전원이 차단되면 기억된 내용이 모두 지워지는 **휘발성(Volatile)** 메모리입니다. 비휘발성 특징을 갖는 것은 ROM입니다.

<br/>

**9. 데이터베이스 관리 시스템(DBMS)이 가지는 기능으로 옳지 않은 것은?**
1. 정의 기능
2. 탐색 기능
3. 조작 기능
4. 제어 기능

> **정답**: 2
> **해설**: DBMS의 3대 필수 기능은 데이터의 구조를 명시하는 **정의(Definition)**, 데이터를 이용하는 **조작(Manipulation)**, 정확성과 무결성을 보호하는 **제어(Control)** 기능입니다. 탐색(검색)은 조작 기능의 하나로 분류됩니다.

<br/>

**10. 팩시밀리의 특징으로 옳지 않은 것은?**
1. 원하는 시간에 원하는 정보 전송이 가능하다.
2. 동일 내용을 한 번에 한 명의 수신자에게만 보낼 수 있다.
3. 일반 전화회선을 이용하여 즉시 전송 가능하다.
4. 종이원고의 내용을 원격지에서 충실하게 기록 재생할 수 있다.

> **정답**: 2
> **해설**: 팩시밀리는 다수의 수신처로 동시 전송 가능한 **동보 전송 기능**이 제공되므로 한 명에게만 전송할 수 있는 것이 아닙니다. 

<br/>

**11. 사무자동화의 특징으로 가장 거리가 먼 것은?**
1. 사무의 생산성 향상
2. 사무처리 시간의 단축
3. 정형적인 업무의 자동화
4. 인건비, 관리비 증가

> **정답**: 4
> **해설**: 기기와 정보시스템을 통한 사무자동화의 주 목표 중 하나는 **간접 비용 및 인건비의 절감**입니다.

<br/>

**12. 자신의 컴퓨터에 전자메일을 선택적으로 내려 받을 수 있도록 할 때 사용하는 프로토콜은?**
1. FTP
2. HTTP
3. SNMP
4. POP3

> **정답**: 4
> **해설**: 메일 서버에 수신되어 보관된 전자우편을 사용자의 로컬 컴퓨터로 가져오는데 사용하는 프로토콜은 **POP3**(Post Office Protocol 3)나 IMAP입니다.

<br/>

**13. 통합 사무자동화 시스템의 개념에 대한 설명으로 알맞지 않은 것은?**
1. 기기 간의 통합은 OA 기기의 공유와 데이터의 공유 등 자원의 공유를 포함한다.
2. OA의 통합화는 복합화된 OA에서 단순 OA로 분리시키는 방향을 의미한다.
3. OA의 통합화는 조직 내에서 개별적으로 진행되던 OA를 일관성 있게 통합함을 의미한다.
4. OA의 통합화는 OA 기기의 통합화와 시스템의 통합화를 기반으로 한다.

> **정답**: 2
> **해설**: 사무자동화(OA)의 통합화 과정은 단순한 개별 기기(단독형 배치)들의 활용에서 **복합, 네트워크화, 시스템 연계를 통한 유기적인 정보 융합** 방향으로 발전합니다. 분리시키는 것이 아닙니다.

<br/>

**14. 사무자동화 추진의 선결 과제로 적합하지 않은 것은?**
1. 추진 목표의 설정
2. 한계선의 설정
3. 전사적 캠페인의 실시
4. 벤치마킹 실시

> **정답**: 4
> **해설**: 자동화를 시작하기 위한 최초 전제조건으로는 목표 수립, 범위(한계선) 확정, 조직 내 공감대 형성(캠페인) 등이 필요합니다. 벤치마킹 분석은 필수 선결 요건이 아닌 구체화 과정의 선택적 방법론입니다.

<br/>

**15. 다음 중 응용 프로그램(Application Program)에 가장 가까운 것은?**
1. 성적 처리 프로그램
2. 정렬/병합 프로그램
3. 제어 프로그램
4. 감시 프로그램

> **정답**: 1
> **해설**: 사용자가 특정 목적(급여, 성적, 인사 등)을 위해 사용하는 소프트웨어가 응용 프로그램입니다. 정렬, 제어, 감시 프로그램은 컴퓨터 시스템 자체를 원활하게 작동하게 돕는 **시스템 소프트웨어** 및 유틸리티 영역에 해당합니다.

<br/>

**16. 계층형 데이터베이스에 대한 설명으로 가장 옳은 것은?**
1. 서로 관계있는 레코드들이 그물처럼 얽혀 있는 구조로 되어 있다.
2. 각 레코드가 트리 구조 형식으로 구성된 모형이다.
3. 수학적 이론에 기초하여 테이블 형태로 표현된 모형이다.
4. 행과 열로 구성된 2차원 구조이다.

> **정답**: 2
> **해설**: 계층형 데이터베이스(Hierarchical Model)는 부모 레코드와 자식 레코드들이 **트리(Tree) 구조**로 연결된 모델입니다. 1번은 망형(네트워크형), 3번과 4번은 관계형 데이터베이스입니다.

<br/>

**17. 정보처리 시스템의 변천 과정이 바르게 나열된 것은?**
1. 비집중처리 시스템 → 집중처리 시스템 → 분산처리 시스템
2. 집중처리 시스템 → 비집중처리 시스템 → 분산처리 시스템
3. 분산처리 시스템 → 집중처리 시스템 → 비집중처리 시스템
4. 비집중처리 시스템 → 분산처리 시스템 → 집중처리 시스템

> **정답**: 1
> **해설**: 초장기 개별적이고 **비집중된 처리** 환경에서, 메인프레임 중심의 **중앙 집중 처리**로 진화하였고, 이후 네트워크 기술 발달로 각 로컬 자원을 융합해 쓰는 **분산 처리 시스템**으로 변천하였습니다.

<br/>

**18. 다음 중 자외선을 이용하여 기억된 내용을 지우고 다시 사용할 수 있는 ROM을 무엇이라 하는가?**
1. Mask ROM
2. PROM
3. EPROM
4. EEPROM

> **정답**: 3
> **해설**: 
- PROM: 1회만 기록 가능
- **EPROM**(Erasable PROM): 자외선을 쬐어 기록을 지우고 갱신 가능
- EEPROM: 전기적인 신호로 지우고 갱신 가능한 플래시 메모리의 기반.

<br/>

**19. 다음 중 사무자동화의 정량적 효과가 아닌 것은?**
1. 수익성 향상을 위한 지원
2. 사무자동화에 따른 전체 비용의 절감
3. 사무원들의 사기 진작에 따른 업무능률 향상
4. 재고, 설비, 금리 등에 대한 고정비, 간접비의 개선

> **정답**: 3
> **해설**: 수치로 증명하기 힘든 직원의 사기 고취나 피로도 감소, 심리적 만족감, 기업 이미지 개선 등은 **정성적(Qualitative) 효과**로 분류됩니다.

<br/>

**20. 자료 송·수신기기(시스템)가 아닌 것은?**
1. 팩시밀리
2. 전자우편
3. EDI
4. 전자출판

> **정답**: 4
> **해설**: 전자출판(DTP, DeskTop Publishing)은 컴퓨터로 문서나 인쇄물을 조판, 편집하고 디자인하는 시스템이므로 원격지와의 송·수진 자체가 본래 목적이 아닙니다.

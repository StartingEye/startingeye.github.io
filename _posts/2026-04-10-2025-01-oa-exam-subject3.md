## 2025년 1회 3과목: 프로그래밍 일반

**41. 실행 가능한 목적 파일을 통합해서 실행하기 위해 메인 메모리에 적재하는 기능을 하는 것은?**
1. 링커
2. 로더
3. 컴파일러
4. 프리프로세서

> **정답**: 2
> **해설**: 주기억장치(메인 메모리)에 프로그램을 **올려놓는(Load) 역할**을 하는 프로그램이 바로 **로더(Loader)**입니다. 목적 파일들을 하나로 묶어주는 것은 링커(Linker)의 역할입니다.

<br/>

**42. 운영체제의 목적으로 거리가 먼 것은?**
1. 처리량 향상
2. 신뢰도 향상
3. 사용 가능도 향상
4. 반환 시간 증대

> **정답**: 4
> **해설**: 운영체제의 목적은 작업이 완료될 때까지 걸리는 **반환 시간(Turnaround Time)을 감소**시키고, 처리량(Throughput), 신뢰도(Reliability), 사용 가능도(Availability)를 높이는 것입니다.

<br/>

**43. C 언어에서 나머지를 구하기 위한 연산자는?**
1. %
2. @
3. #
4. !

> **정답**: 1
> **해설**: 대부분의 프로그래밍 언어에서 나눗셈의 **나머지를 구하는 모듈러 연산자** 기호는 `%`를 사용합니다.

<br/>

**44. 객체지향 프로그램의 주요 개념이 아닌 것은?**
1. 고정 자료형(Fixed Data Type)
2. 추상 데이터형(Abstract Data Type)
3. 자료 은폐(Data Hiding)
4. 상속형(Inheritance)

> **정답**: 1
> **해설**: 객체지향의 핵심 특징은 **추상화(캡슐화), 정보 은폐, 상속, 다형성** 등입니다. 고정 자료형은 구조적/절차적 언어의 특징이며, 객체지향에서는 유연한 추상 데이터형을 중시합니다.

<br/>

**45. 클래스 간 계층 관계에 근거하고 클래스 간 속성과 연산을 공유하는 객체지향 언어의 특징은?**
1. 다형성
2. 객체
3. 추상화
4. 상속성

> **정답**: 4
> **해설**: **상속성(Inheritance)**은 상위(부모) 클래스에 정의된 속성과 메서드를 하위(자식) 클래스가 물려받아 재사용하거나 확장할 수 있게 하는 기능입니다.

<br/>

**46. 어휘 분석 단계에 대해 적절하게 표현한 것은?**
1. 구문 분석이 끝난 후에 실행한다.
2. 원시 프로그램을 읽어 처리하는 단계이다.
3. 일명 파싱이라고 부른다.
4. 구문 단위를 형성하는 어휘 항목을 식별한다.

> **정답**: 4
> **해설**: 컴파일러의 첫 단계인 어휘 분석(Lexical Analysis)은 소스 코드를 읽고 문법적인 의미를 지니는 최소 단위인 토큰(Token) 등을 **어휘 항목으로 식별**하는 과정입니다.

<br/>

**47. 기계어에 대한 설명으로 옳지 않은 것은?**
1. 2진수 0과 1만 사용하여 명령어와 데이터를 나타낸다.
2. 컴퓨터가 직접 이해할 수 있어 실행 속도가 빠르다.
3. 전문적인 지식이 없으면 이해하기 힘들다.
4. 모든 기계에서 공통으로 사용 가능하여 호환성이 높다.

> **정답**: 4
> **해설**: 기계어는 각 CPU 아키텍처나 제조사마다 명령어 집합(Instruction Set)이 다르므로, 특정 기계에 종속적이며 **기기 간 호환성이 매우 낮습니다**.

<br/>

**48. EBNF 표기법에서 반복을 나타내는 기호는?**
1. ::=
2. |
3. <>
4. { }

> **정답**: 4
> **해설**: 메타 언어 표기법인 EBNF(Extended BNF)에서 **중괄호 `{ }`**는 해당 요소가 0번 이상 반복될 수 있음을 나타냅니다.

<br/>

**49. Java에서 하위 클래스에서 상위 클래스를 참조하기 위해 사용하는 명령어는?**
1. Extends
2. Static
3. Super
4. Method

> **정답**: 3
> **해설**: 자바(Java)에서 자식 클래스가 부모 클래스의 생성자나 변수, 메서드를 참조할 때 사용하는 키워드는 **super**입니다. `extends`는 클래스를 상속받겠다고 선언할 때 사용합니다.

<br/>

**50. 10진수 937를 BCD 코드로 표현한 것은?**
1. 100100110111
2. 001100110100
3. 011000110011
4. 001100110011

> **정답**: 1
> **해설**: BCD(2진화 10진) 코드는 10진수 각 자리를 4비트 2진수로 풀어서 표현합니다.
- `9` -> 1001
- `3` -> 0011
- `7` -> 0111
이어 붙이면 **`100100110111`**이 됩니다.

<br/>

**51. 특수 형태의 토큰에 해당되지 않는 것은?**
1. 지정어
2. 연산자
3. 상수
4. 구분자

> **정답**: 3
> **해설**: 컴파일러 구문 분석의 토큰(Token) 종류 중 특수 기호 형태로 취급되는 것은 구두점(구분자), 괄호, 연산자 등입니다. **상수(Literal/Constant)**나 식별자는 일반적인 어휘 항목으로 취급됩니다.

<br/>

**52. 프로그램 개발 과정에서 프로그램 안에 내재해 있는 논리적 오류를 발견하고 수정하는 작업은?**
1. Loading
2. Debugging
3. Linking
4. Hashing

> **정답**: 2
> **해설**: 소프트웨어의 논리적인 버그(오류)를 찾아내 고치는 과정을 탈레를 잡는다는 뜻의 **디버깅(Debugging)**이라고 부릅니다.

<br/>

**53. C 언어에서 사용하는 기억 클래스에 해당하지 않은 것은?**
1. Static
2. Register
3. Internal
4. Auto

> **정답**: 3
> **해설**: C 언어의 변수 기억 역영(Storage Class) 종류로는 크게 **auto(자동), register(레지스터), static(정적), extern(외부)**이 있습니다. 'Internal'이라는 지시어는 없습니다.

<br/>

**54. "A＋B＊C－D"를 후위(Postfix) 표기법으로 표현한 것은?**
1. A B C ＊ D － ＋
2. A B ＋ C ＊ D －
3. A B C ＋ ＊ D －
4. A B C ＊ ＋ D －

> **정답**: 4
> **해설**: 연산자 우선순위는 곱셈(`*`) -> 덧셈/뺄셈(`+`, `-`) 순서입니다.
1. `B*C` -> `BC*`
2. `A+(BC*)` -> `ABC*+`
3. `(ABC*+) - D` -> **`ABC*+D-`**

<br/>

**55. 프로그래밍 언어에서 예약어란?**
1. 프로그래머가 미리 설정한 변수
2. 데이터를 저장할 수 있는 이름이 부여된 기억 장소
3. 시스템이 알고 있는 특수한 기능을 수행하도록 이미 용도가 정해져 있는 단어
4. 프로그램이 수행되는 동안 변하지 않는 값을 나타내는 단어

> **정답**: 3
> **해설**: `if`, `while`, `int` 등과 같이 컴파일러가 처리 방법을 이미 정해두고 **특수한 문법적 목적으로 쓰기로 예약한 단어**들을 예약어(Reserved word)라고 합니다. 사용자가 변수 이름 등으로 쓸 수 없습니다.

<br/>

**56. C 언어의 이스케이프 시퀸스에서 "carriage return"을 나타내는 기호는?**
1. \t
2. \n
3. \b
4. \r

> **정답**: 4
> **해설**: **`\r`** 은 Carriage Return(커서를 현재 줄의 맨 앞으로 이동)을 의미합니다. `\n`은 줄바꿈(Line Feed), `\t`는 탭 공간입니다.

<br/>

**57. C 언어에서 ‘char’은 어떤 변수를 선언할 때 사용되는 예약어인가?**
1. 정수형
2. 실수형
3. 문자형
4. 배정도

> **정답**: 3
> **해설**: **char**는 Character의 약자로, 1바이트 크기의 **문자형(문자 데이터)** 변수를 선언할 때 사용합니다. 정수형은 `int`, 실수형은 `float`, `double`을 씁니다.

<br/>

**58. 아래 코드의 실행 결과는?**
```c
#include <stdio.h>
int main(void) {
    int code = 65;
    int *p = &code;
    printf("%c", (*p)++);
}
```
1. A
2. B
3. 65
4. 66

> **정답**: 1
> **해설**: `*p`는 65라는 값을 가리킵니다. 아스키코드로 65는 소문자 'A'가 아닌 대문자 **'A'**입니다. 후위 증감 연산자 `++`가 쓰였으므로 `printf`에는 먼저 65('A')가 출력되고 나서, 다음에 값이 66으로 증가합니다.

<br/>

**59. 다음의 C 언어 연산자 기호 중에서 우선순위가 가장 먼저인 것은?**
1. && (논리AND)
2. == (관계비교)
3. = (대입)
4. / (나눗셈 산술)

> **정답**: 4
> **해설**: 보통 프로그래밍 연산자는 **산술 연산자(`*`, `/`) -> 관계 연산자(`<`, `==`) -> 논리 연산자(`&&`) -> 대입 연산자(`=`)** 순으로 우선순위를 갖습니다.

<br/>

**60. 교착 상태의 발생 조건이 아닌 것은?**
1. 상호 배제
2. 환형 대기
3. 선점
4. 점유와 대기

> **정답**: 3
> **해설**: 교착 상태(Deadlock)의 발생 4대 조건은 상호 배제(Mutual Exclusion), 점유와 대기(Hold & Wait), '비선점(Non-preemption)' 그리고 환형 대기(Circular Wait)입니다. 권한을 빼앗을 수 있는 **선점** 조건이나 환경에서는 교착 상태를 해결할 수 있습니다.
